Writing for Hackers

The Story

This project began as gist.io, a wonderfully simple service created by Idan Gazit that let you share GitHub Gists as beautifully-rendered Markdown pages. The original was a Node.js application that delighted developers who wanted zero-setup publishing.

When it came time to breathe new life into it, we translated the whole thing into a warm, self-contained Go application. The server is pure Go, and Markdown is rendered right in the browser via a tiny WebAssembly module — also written in Go — so there is no external render service to depend on.

The result is goRenderMand: the same spirit of effortless Markdown publishing, now running on a fast, single-binary Go backend. Paste a URL, get a beautifully-formatted page.

Render any public Markdown URL

GitHub Gist shortcut

  1. Create a gist on GitHub with a single Markdown-syntax file.
  2. In the URL bar, replace
    gist.github.com/youruser/abc123… with
    gist.io/@youruser/abc123….
  3. Enjoy your beautifully-presented writing.
